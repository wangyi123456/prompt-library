# Prompt Library

一个用于整理和维护高质量提示词的个人仓库。

## 目录结构

```text
prompts/
  daily-brief/       每日简报类提示词
  research/          研究与信息检索
  writing/           写作与编辑
  coding/            编程与工程
  productivity/      效率与工作流

templates/           Prompt 模板
```

## 收录规范

每个 Prompt 建议包含：

- 标题
- 使用场景
- Prompt 正文
- 备注或维护记录
